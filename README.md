# HTML5-Video

Розроблено веб-плейер на HTML5 що має такі функціональні можливості:

  - Змінення гучності
  - Повноекранний режим
  - Buffering
  - Прокручування відео
  - Зміна відео
  - Підрахунок часу перегляду і загального часу
  - Попередній перегляд відео
